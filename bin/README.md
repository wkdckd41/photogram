# photogram
Springboot + JPA를 이용한 SNS 웹사이트 프로젝트

1. 프로젝트 소개

- AOP를 활용한 유효성 검사 자동화
- 커스텀 예외 핸들링
- 복잡한 모델 DTO 매핑
- 스프링 시큐리티
- OAuth2.0
- 복잡한 연관관계 처리
- N:N 처리
- ResponseEntity 사용

2. 프로젝트 환경 구성

- JDK 11
- MariaDB 10.7
- Springboot2.6.6
- AOP
- QLRM
- JSP Template Engine
- Maven
- Lombok
- Springboot JPA
- Springboot Security
